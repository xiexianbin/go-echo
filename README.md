## usage

- release

```
git tag -s v0.1.0
git push origin v0.1.0
# git push origin --tags
```

- download

```
curl -Lfs -o main https://github.com/xiexianbin/go-echo-demo/releases/latest/download/main-{linux|darwin|windows}-{amd64|arm64}
chmod +x main
./main
```
